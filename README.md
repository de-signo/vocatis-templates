## Getting Started

Requirements:

- nodejs v18
- npm 8

Build the project:

```
npm install
cd 5.0
npx gulp
```

Use the files in the `dist` folder to install in Vocatis.

After building the project one, you're able to view the templates locally using `ng serve`. For example:

```
cd 5.0/printer
ng serve printer
```
